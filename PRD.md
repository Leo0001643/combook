# CamBook 上门按摩SPA平台 — 完整产品需求文档 (PRD)

**版本**: V1.0.0  
**日期**: 2026-04-12  
**产品定位**: 东南亚（柬埔寨、越南）及国际市场的上门按摩SPA预约服务平台

---

## 一、产品概述

CamBook 是一款连接用户与专业按摩技师的O2O上门服务平台，支持用户在线预约技师上门提供专业按摩/SPA服务，覆盖柬埔寨、越南等东南亚市场，兼容中文、英文、越南文、柬埔寨文四种语言。

---

## 二、用户角色

| 角色 | 说明 |
|------|------|
| 会员（Member） | 普通用户，注册后可浏览技师、下单预约、支付、评价 |
| 技师（Technician） | 注册后提交资质认证，接受订单、上门服务、收款 |
| 商户（Merchant） | 管理旗下技师团队，查看收入报表，运营活动 |
| 管理员（Admin） | 全平台管理、审核、数据报表、配置 |

## 四、非功能性需求

### 4.1 多语言支持
| 语言 | 代码 | 说明 |
|------|------|------|
| 中文简体 | zh-CN | 全局覆盖 |
| English | en | 全局覆盖 |
| Tiếng Việt | vi | 全局覆盖 |
| ភាសាខ្មែរ | km | 全局覆盖 |

### 4.2 支付方式
| 支付方式 | 说明 |
|---------|------|
| USDT (TRC20/ERC20) | 加密货币，面向国际用户 |
| ABA Bank Transfer | 柬埔寨 ABA 网银 |
| 平台余额 | 充值后使用 |

### 4.3 地图与定位
- 使用 Google Maps Platform
- 用户定位：实时GPS定位
- 技师定位：WebSocket 实时上报
- 距离计算：Haversine 算法
- 导航：Google Maps Navigation

### 4.4 验证码
- 短信验证码：支持柬埔寨 (+855) 手机号
- 服务商：Twilio / AWS SNS（支持柬埔寨区号）
- 邮箱验证码备用

### 4.5 推送通知
- Firebase Cloud Messaging (FCM)
- 新订单推送、状态变更推送

### 4.6 IM 即时通讯
- WebSocket 实现
- 消息持久化
- 离线消息推送

### 4.7 安全
- JWT Token 认证（双 Token：AccessToken + RefreshToken）
- 接口幂等性
- 防 XSS / SQL 注入
- 敏感数据加密存储
- OSS 图片上传（AWS S3 / 阿里云 OSS）

## 五、技术架构

### 后端 (cambook-server)
```
JDK 17 + Spring Boot 3.x
MyBatis-Plus 3.x
MySQL 8.0（主从复制）
Redis 7.x（缓存/Session/分布式锁）
RocketMQ（异步消息）
WebSocket（IM/定位）
Spring Security + JWT
Swagger 3 (OpenAPI)
```

### 管理前端 (cambook-admin)
```
React 18 + TypeScript
Ant Design Pro 6.x
Vite
Zustand（状态管理）
React Router 6
Axios
ECharts（图表）
```

### 移动端 (cambook_app)
```
Flutter 3.x（Dart）
BLoC + Cubit 状态管理
go_router（路由）
dio（HTTP）
google_maps_flutter（地图）
flutter_localizations（多语言）
web_socket_channel（WebSocket）
firebase_messaging（推送）

```

- `technician_location` — 技师位置
- `user_address` — 用户地址
- `review` — 评价
- `im_message` — IM消息
- `im_session` — IM会话
- `push_notification` — 推送记录
- `banner` — 轮播图
- `sys_config` — 系统配置
